🚀 QUICK START - EXACTLY WHAT TO RUN:

Step 1: Install globally
    npm install -g .

Step 2: Setup (first time)
    jarvis init

Step 3: Use it
    jarvis list-skills
    jarvis run-skill --skill ./skills/example.yml

That's it! You can now use 'jarvis' from anywhere on your device.

💻 COMMON COMMANDS:

    jarvis init                              # Setup wizard

    jarvis run-skill --skill path/to/skill   # Run a skill
    jarvis speak --text "Hello"              # Text to speech
    jarvis listen --file audio.wav           # Speech to text
    jarvis --help                            # Show all commands
